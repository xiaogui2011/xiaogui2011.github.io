## 清理不必要的文件

目的: 释放一部分空间, 方便登录界面

```
find ./ "*.log" -type f -exec rm {} \;
```

```
find ./ -name "*.tmp" -type f -exec rm {} \;
```

```
find -name "cron.log" -type f -mtime +7 -exec rm {} \;
```

```
find ./ -size +100M -type f
```
